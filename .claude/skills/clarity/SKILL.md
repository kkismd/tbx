## 使用方法

```
/clarity <ファイルパス>
```

`<ファイルパス>` にはレビュー対象ファイルのパスを指定する（リポジトリルートからの相対パス、または絶対パス）。

## 動作

新しいサブエージェント（`general-purpose`）を起動し、以下を依頼する：

```
<ファイルパス>（リポジトリ: /home/cake/src/Rust/tbx）を読んで、不明瞭なところ、自分の解釈や判断で補う必要がありそうな部分があれば教えてください。
優先度（高・中・低）をつけて箇条書きで報告してください。
```

## ポイント

- **新しいサブエージェント**を起動することで、先入観のないフレッシュな目でファイルを読ませる
- 報告された指摘をもとに修正するかどうかはユーザーが判断する
- 指摘がなくなるまで繰り返すと際限がなくなるため、高・中優先度の指摘が解消されたらマージを検討する
